# Генератор КП — Smart Restaurant

Веб-приложение для быстрой генерации брендованных коммерческих предложений на 2 страницы для партнёров Smart Restaurant.

Заполняешь форму слева → справа сразу видишь готовое КП → печатаешь в PDF через браузер.

## Возможности

- Live-preview во время заполнения формы.
- 5 продуктов на выбор с редактируемой ценой: SR Delivery, Авто-подтягивание счёта (QR-оплата в зале), Без Официанта (QR-меню), Лояльность, Без Кассира.
- Автоматические расчёты экономики доставки: комиссия агрегаторов, чистый эффект Delivery, сценарии 30 / 50 / 70 / 100 %.
- Условные блоки: показываются только выбранные продукты.
- Экспорт в PDF через `Cmd/Ctrl + P` или кнопку «Печать в PDF».
- Скачивание исходного HTML.
- Сохранение последних настроек в `localStorage` — открываешь файл и форма уже заполнена.
- Полностью статичный сайт, без сервера и сборки. Один HTML-файл.

## Запуск

### Локально

```bash
git clone https://github.com/<твой-юзер>/kp-generator.git
cd kp-generator
open index.html
```

Или просто двойной клик по `index.html` — откроется в браузере.

### GitHub Pages

1. Сделай fork репозитория или запушь свою копию.
2. В репозитории: **Settings → Pages → Source: Deploy from a branch → main / root → Save**.
3. Через минуту приложение будет доступно по адресу `https://<твой-юзер>.github.io/kp-generator/`.

## Кастомизация

- **Цены продуктов** — редактируются прямо в форме рядом с чекбоксом.
- **Цвета бренда** — переменные `--sr-green*` в `<style>` блока `:root` в начале `index.html`.
- **Текст буллетов** про продукты — в JS-объекте `PRODUCT_BULLETS` ближе к концу файла.
- **Контакты менеджера** — в форме слева.
